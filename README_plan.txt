# Main README
## Dev
в голове сейчас что-то такое:

 - это будет сервис, гоняющийся на google cloud. 
 - В основе это будет веб-приложение на основе flask, которое будет уметь общаться с телеграмом и дальше реализовывать часть "бота"
 - К этому всему нужна какая-нибудь SQL (или векторная, для пущего пафоса) датабаза, в которой будет хранится словарь. 
 - В словаре будут, непосредственно, слова изучаемого языка, их "уровень" (какое-нибудь число от 0 до 1)
 - дальше к этому присобачить какие-то user-dependant metrics - а-ля, насколько это слово релевантно и его надо повторить, как часто юзер в нем фейлит итд
